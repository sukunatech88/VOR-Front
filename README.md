# VOR Web

Frontend enterprise inicial de VOR Phase 1.

## Stack
- React
- TypeScript
- Vite
- React Router
- TanStack Query
- Tailwind CSS
- Zod

## Estado actual
Base del frontend en construcción desde cero.

## Objetivo
Preparar una SPA modular, limpia y escalable para:
- Dashboard
- File Registry
- Message Hub
- User Administration
- futuros módulos operativos y de auditoría
